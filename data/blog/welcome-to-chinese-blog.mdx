## 关于这个博客

这个博客是基于 [Next.js](https://nextjs.org/) 和 [Tailwind CSS](https://tailwindcss.com/) 构建的，基于优秀的开源模板 [tailwind-nextjs-starter-blog](https://github.com/timlrx/tailwind-nextjs-starter-blog)进行二次开发。

### 主要特性

- ✨ **现代化技术栈**：Next.js 15 + React 19 + TypeScript
- 🎨 **优雅的设计**：Tailwind CSS + 深色/浅色主题切换
- 📝 **MDX 支持**：在 Markdown 中使用 React 组件
- 🔍 **搜索功能**：内置搜索，快速找到内容
- 📱 **响应式设计**：完美适配各种设备
- ⚡ **性能优化**：接近满分的 Lighthouse 评分
