# 视觉识别与设计系统提案

## 1. 项目定位："个人数字花园"

**身份**：个人与真实（非商业化）。我们正在构建一个"数字花园"或"工坊"——一个让思想生长和展示的空间，而不是一个转化潜在客户的店面。

- **目标氛围**：沉思、宁静、欢迎、知性且平易近人。
- **与用户的关系**：界面像是一个邀请客人进入客厅的主人，而不是一个试图最大化转化率的销售员。

## 2. 设计哲学："温暖有机极简主义" (Warm Organic Minimalism)

这种风格在实用与人文温暖之间架起桥梁。它去除杂乱，但保留灵魂。

### 核心原则

1.  **触感胜于光泽**：界面应感觉像纸张、画布或石头，而不是玻璃或塑料。
2.  **排版即主角**：内容为王。字体应当具有个性。
3.  **源于自然的色彩**：颜色源自自然颜料（粘土、树叶、石头、天空），而非 RGB 屏幕原色。
4.  **不完美之美**：承认"手工感"。完美的对称属于机器；轻微的有机不对称或柔和感属于人类。

### ❌ 严格避免的风格

- **企业级 SaaS**：不要"仪表盘蓝"，不要密集的数据表格，不要"立即升级"徽章。
- **营销噱头**：不要巨大的渐变，不要漂浮的 3D 等轴插图。
- **科技冷感**：不要纯黑(#000000)配纯白(#FFFFFF)，不要"黑客帝国绿"，不要"赛博朋克"霓虹。
- **冷色渐变**：不要紫色到蓝色的褪色效果（常见的 Linear 风格 SaaS 感）。

## 3. 视觉语言与 Token

### A. 配色方案 (提议主题)

我们将使用 **OKLCH** 以获得感知均匀的颜色。

#### 选项 1："晨雾与苔藓" (沉静、知性)

_核心：深绿色、暖灰色、未漂白的纸张。_

- **背景**: `oklch(0.99 0.005 100)` (暖米白 / 纸张色)
- **表面/卡片**: `oklch(0.97 0.01 100)` (稍暗的纸张色)
- **前景 (文字)**: `oklch(0.30 0.02 100)` (深暖石灰)
- **主色**: `oklch(0.55 0.15 140)` (有机叶绿)
- **强调色**: `oklch(0.70 0.12 60)` (柔和焦橙色)

#### 选项 2："陶土与画布" (温暖、艺术)

_核心：大地色调、赤陶色、奶油色背景。_

- **背景**: `oklch(0.98 0.01 90)` (奶油 / 亚麻)
- **前景 (文字)**: `oklch(0.25 0.03 50)` (深大地褐)
- **主色**: `oklch(0.60 0.18 45)` (赤陶 / 砖红)
- **次要色**: `oklch(0.85 0.10 85)` (沙色)

_(建议：我们从 **选项 1**
开始，因为它在可读性上更安全，但会对其进行微调，使其比当前的祖母绿更温暖)_

### B. 排版 (Typography)

为了实现"艺术与手工感"：

- **标题 (Headings)**：衬线字体 (Serif) 是必不可少的。
  - _建议_：**"Cinzel"** (典雅) 或 **"Lora"** (书法/艺术感) 或 **"Libre
    Baskerville"** (传统)。
  - _当前_："Geist"
    (Sans)。我们应保留 Geist/Manrope 用于 UI 文本，但使用 Serif 用于内容标题。
- **正文 (Body)**：人文主义无衬线体 (Humanist Sans)。
  - _当前_：Manrope 很棒。它是几何的，但有柔和的曲线。

### C. 形状与空间

- **圆角 (Radius)**：`--radius: 0.75rem` (12px) 或 `1rem` (16px)。比标准 SaaS
  (4-8px) 更柔和的角落。
- **间距 (Spacing)**：宽裕、"透气"。默认为 `gap-6` 或 `gap-8`。
- **布局**：
  - 文章采用单栏阅读体验。
  - 索引页采用瀑布流或不规则网格（艺术感）。

## 4. 视觉元素与动效

- **按钮**：
  - 避免"闪亮按钮"。使用"哑光"外观。
  - 微妙的颗粒纹理叠加？(进阶)。
  - 悬停：颜色偏移，而不是阴影提升。
- **卡片**：
  - **无阴影 / 仅边框**：看起来像草图。
  - **或 柔和漫射阴影**：`box-shadow: 0 10px 40px -10px rgba(0,0,0,0.05)`
    (非常柔和)。
- **动画**：
  - **"呼吸感"**：元素缓慢淡入并上浮 (`duration-700 ease-out`)。
  - **悬停**：缓慢过渡 (`duration-300`)。

## 6. UI 设计风格细则： "纸张与墨水" (Paper & Ink)

我们的 UI 应当感觉像是高质量的印刷品，而非发光的屏幕。

### A. 材质与层级 (Materiality & Elevation)

- **拒绝玻璃拟态 (Glassmorphism)**：不要使用模糊的背景。那是 iOS/Tech 的风格。
- **纸张隐喻**：
  - **背景层**：粗糙的纸张纹理（纯色或极微噪点）。
  - **内容层**：浮在背景上的卡片，不是通过阴影，而是通过**边框**或**微妙的色差**来区分。
  - **层级**：尽量扁平。只有在模态框 (Modal) 或下拉菜单 (Dropdown) 时才使用显著阴影。

### B. 按钮与控件 (Buttons & Controls)

- **主要按钮 (Primary)**：
  - _外观_：实心填色，无渐变。由于是"有机"风格，边框可以有轻微的 `border-radius`
    不一致（如果技术允许）或者保持大的圆角（Pill shape 或 Soft Rect）。
  - _触感_：按下时缩小 (Scale down 0.98)，而不是变色。
- **次要按钮 (Secondary)**：
  - _外观_：透明背景，深色边框 (1px border)，文字颜色与边框一致。
- **输入框 (Inputs)**：
  - _未激活_：只有底部边框 (Underline) 或 极淡的全包围边框。看起来像填空题的横线。
  - _激活_：边框变深，背景微亮。不要原本的蓝色光晕 (Focus
    Ring)，改为深灰色或品牌色的双层边框。

### C. 导航 (Navigation)

- **顶部导航**：透明或与背景同色。由于内容为王，导航应在滚动时隐藏或缩小，不要一直占据视觉重心。
- **面包屑**：使用衬线字体斜体显示，作为一种装饰性元素。

## 7. 动画与交互风格： "风吹过花园" (Breeze in the Garden)

动画应当通过缓动 (Easing) 来传达重量感和自然感。

### A. 核心动效曲线 (The Curves)

- **避免**：`linear` (线性), `ease-in-out` (标准的正弦波)。
- **采用**：
  - **进入**：`cubic-bezier(0.2, 0.0, 0.2, 1.0)`
    (快速进入，极慢且平滑的停止)。感觉像叶子飘落着地。
  - **退出**：`cubic-bezier(0.4, 0.0, 1.0, 1.0)` (加速离开)。

### B. 页面过渡 (Page Transitions)

- 不要整页"左滑/右滑" (太像 PPT)。
- **内容淡入上浮**：新页面加载时，标题先固定，内容从下方 20px 处缓慢淡入 (`opacity: 0 -> 1`,
  `transform: translateY(20px) -> 0`)，持续时间 `0.6s` ~ `0.8s`。
- 这创造了一种"浮现"的宁静感。

### C. 微交互 (Micro-interactions)

- **链接悬停**：
  - 不要简单的下划线出现。
  - 尝试：背景色高亮（就像用荧光笔画过），或者左侧出现一个小图标。
- **图片加载**：
  - 不要瞬间弹出。
  - 使用模糊到清晰 (Blur-up) + 灰度到彩色 (Grayscale to
    Color) 的过渡。这符合"显影"的胶片感。
